要求

2.7

油浸

部件

C

(1)油浸型电气部件应符合本规则2.1项和2.2 项相应的防爆要求；

(2)油浸型电气部件必须密封良好，不允许渗漏油，油位高度在规定范围内；外壳、电气和机械连接所用的螺栓、螺母以及注油、排油的螺栓塞等均需有防松措施

9

2.8

正压

机房

C

(2)正压型机房通风充气系统，应有先通风后供电，先停电后停风的气电联锁装置。通风电气部件如安装在非正压型机房内，则应符合本规则2.1项和2.2项相应的防爆要求；

(3)正压型机房微差压继电器应装在风压、气压最低点的出口处；运行中的机房内气压低于设计要求时，微差压继电器应可靠动作

2.9

防爆

接线

盒C

(1)敷设在防爆区域内非本安电路的电缆不得直接连接，如必须连接或分路时，应设置防爆接线盒；

(2)防爆接线盒应符合本规则2.1项和2.2项相应的防爆要求

11

2.10

电缆

配线

C

(1)防爆区域内应当采用橡胶电缆或者铠装电缆配线；

(3)电缆上易发生机械损伤部位的电缆应采取防机械损伤的保护措施

12

2.11

本安

配线

C

(1)本安电路的电缆或电线以及防护套管至少在进出端部应设有浅蓝色标识；

(3)本安电路与非本安电路在同一个接线箱内连接时需有绝缘板分隔或者间距应大于50mm

2.12

电缆

引入

C

非本安型防爆电气部件应采用电缆引入装置(密封方法为弹性密封圈或填料)，该装置应该能够夹紧电缆，夹紧组件可以通过夹紧措施、密封圈或者填料来实现

2.13

防爆

封堵

C

非本安型防爆电气部件外壳上多余的电缆引入孔应用符合出厂要求的封堵件封堵
